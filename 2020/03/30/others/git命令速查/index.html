统一概念：

  • 工作区：改动（增删文件和内容）
  • 暂存区：输入命令：git add改动的文件名，此次改动就放到了暂存区
  • 本地仓库(简称：本地)：输入命令：git commit 此次修改的描述，此次改动就放到了本地仓库，每个commit，我叫它为一个版本。
  • 远程仓库(简称：远程)：输入命令：git push 远程仓库，此次改动就放到了远程仓库（GitHub 等)
